bool Si7013_Detect ( I2C_TypeDef *  i2c,
uint8_t  addr,
uint8_t *  deviceId 
)

Checks if a Si7013 is present on the I2C bus or not.

Parameters
[in]i2cThe I2C peripheral to use (Not used).
[in]addrThe I2C address to probe.
[out]deviceIdWrite device ID from SNB_3 if device reponds. Pass in NULL to discard. Should be 0x0D for Si7013, 0x14 for Si7020 or 0x15 for Si7021
Returns
True if a Si7013 is detected, false otherwise.
